Member Function Documentation

void PhotoModel::appendSavedPosition(double lati, double longi)

Ajoute une entrée spéciale dans le Modèle, correspondant à une position GPS mémorisée (marker jaune).

lati : latitude au format GPS. longi : longitude au format GPS.

void PhotoModel::duplicateData(int row)

Duplicates an item of the model, and add it at the end of the vector. row : item row to be duplicated.

void PhotoModel::fetchExifMetadata(int photo = -1)

Ce slot lit des données EXIF de toutes photos du répertoire. photo : l'indice de la photo (vide ou -1 = toutes les photos du répertoire)

void PhotoModel::removeData(int row)

Fonction typique qui supprime l'item désigné du modèle. row : la position dans le vecteur de l'item à modifier.

void PhotoModel::removeSavedPosition()

Supprime du modèle l'item correspondant à la position sauvegardée.

void PhotoModel::saveMetadata()

Ce slot enregistre dans le fichier JPG les metadonnées IPTC qui ont été modifiées. Tags non-modifiés: Make, Model, Tags modifiés: coords GPS, Artist.

void PhotoModel::setData(int row, int value, int property)

Ce slot ajoute une propriété à une photo, par exemple si on clique sur une suggestion.

row : indice de la photo. value : valeur de la propriété. property : nom de la propriété.

void PhotoModel::setInCircleItemCoords(double latitude, double longitude)

Affecte les coordonnées GPS fournies à toutes les photos géographiquement situées à l'interieur du cercle rouge. latitude au format GPS. longitude au format GPS.
