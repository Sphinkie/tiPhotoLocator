Detailed Description

La classe ExifWriteTask permet d'écrire des metadata dans les photos JEG de façon asynchrone.

\inmodule TiPhotoLocator

Tache asynchrone par utilisation de QThreadPool.

Note
: les QRunnable n'héritent pas de QObject et ne peuvent donc pas communiquer avec les autres objets à l'aide de signaux. Donc, à la fin du traitement, pour actualiser les données du PhotoModel, il faut faire un appel direct. Cependant, cela n'est pas contraire aux recommandations: mettre à jour des données peut se faire par appel synchrone.
