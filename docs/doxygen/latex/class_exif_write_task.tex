\doxysubsection{Detailed Description}
La classe \doxylink{class_exif_write_task}{Exif\+Write\+Task} permet d\textquotesingle{}écrire des metadata dans les photos JEG de façon asynchrone. 

\textbackslash{}inmodule Ti\+Photo\+Locator 

Tache asynchrone par utilisation de QThread\+Pool.

\begin{DoxyNote}{Note}
\+: les QRunnable n\textquotesingle{}héritent pas de QObject et ne peuvent donc pas communiquer avec les autres objets à l\textquotesingle{}aide de signaux. Donc, à la fin du traitement, pour actualiser les données du \doxylink{class_photo_model}{Photo\+Model}, il faut faire un appel direct. Cependant, cela n\textquotesingle{}est pas contraire aux recommandations\+: mettre à jour des données peut se faire par appel synchrone. 
\end{DoxyNote}
